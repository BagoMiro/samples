Before you start read these tips:

0. When generating images your browser will freeze, but don't worry, this is all part of the plan (it will make generating faster).
1. Use low resolution otherwise generating text-pixel picture might take a long time and your browser might freeze for few minutes.
2. Larger sampling size creates smaller and not exactly accurate text-pixel images, lower it if you are not satisfied with result it might change a lot.
3. Color images generated contain HTML code therefore they are mostly unusable, but still cool to look at.
4. Large text-pixel images may not be copied through the button because of they huge size so be carefull not to waste your time generating them.
5. If there is discoloration on bottom or right side of the generated image try to resize canvas by few pixels.
6. Analyzer is in experimental stage it might improve visibility of some images though won't really do much usually.
